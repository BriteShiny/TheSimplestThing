Linking is the essence of HTML. The very acronym stands for Hypertext Markup Language. What is hypertext? According to the W3C, "hypertext is text which contains links to other texts". We are so used to this concept by now we don't think much about it. But it was revolutionary. And it is still critical to understanding web development.

Navigation is the first principle of the web.
