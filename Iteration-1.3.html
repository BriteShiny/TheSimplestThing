So how is a link created? By enclosing the text in an Anchor tag. Following is how the source of the link looks:

<a href="Iteration-1.1.html">Navigation</a>

HTML is comprised of tags, or elements, that determine the behavior and rendering of the page. This is the Markup Language part of the HTML acronym, which stands for(Hypertext Markup Language). You can see that the anchor tag is created by wrapping the markup between a less than (<) and a greater than (>) symbol. The markup itself consists of identifying the element "a" and its attributes, in this case href (hypertext reference), which indicates where the link should navigate to. The link on the Anchor tag opens W3C Schools, I'm not going to go through an explanation Hypertext Markup Language. What is hypertext? According to the W3C, "hypertext is text which contains links to other texts". We are so used to this concept by now we don't think much about it. But it was revolutionary. And it is still critical to understanding web development.

Navigation is the first principle of the web.
